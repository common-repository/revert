== Description ==

A one-click button to revert a post to its previous revision. Handy for when you (or your clients) make a mess and want to revert without going through the revisions screen.

== Installation ==

1. Unzip the ZIP file and drop the folder straight into your wp-content/plugins directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Edit any post type that supports revisions and you'll see a link above the Publish/Update button to revert to the previous revision if there is one.

== Frequently Asked Questions ==

= Why hasn't my reverted post shown up? =

You still need to save the post after reverting it. The reverted post is loaded into the editor ready for you to continue editing and save.
